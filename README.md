
# Todo List
Todo-List é um agendador de tarefas, criado em java afim de estudos para um projeto de um processo seletivo

## Requisitos

Java 8 posterior


## Rodando o Projeto

Clone o projeto

```bash
  git clone https://link-para-o-projeto
```

Entre no diretório do projeto

```bash
  cd my-project
```

Compile o codigo

```bash
  javac -cp . App.java
```

Inicie o programa

```bash
  java App
```
